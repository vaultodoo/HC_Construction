Material Requisition for Project and Task Integration Odoo App

Task and Project on Material Purchase Requisition Request by Employee in Odoo

This module develops to Integration of Material Requisition for Project and Task / Job Order. This app allows your user to select project and task on material requisition request form. Using this module your user can create material requisitions for projects and tasks they are working on.

Main Features

  • Added Project and Task on Material Purchase Requisition Request form.
  • Added Material Requisition Smart Button on Project and Project Task.
  • Group by Project and Task on Material Purchase Requisitions.
